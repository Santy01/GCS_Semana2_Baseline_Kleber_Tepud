# Tests

Este directorio contiene las pruebas del sistema.

## Estructura de Pruebas

```
tests/
├── unit/           # Pruebas unitarias
├── integration/    # Pruebas de integración
├── e2e/           # Pruebas end-to-end
└── README.md      # Este archivo
```

## Tipos de Pruebas

### Pruebas Unitarias
Pruebas de componentes individuales y funciones aisladas.

### Pruebas de Integración
Pruebas de interacción entre componentes.

### Pruebas End-to-End
Pruebas del flujo completo de usuario.
